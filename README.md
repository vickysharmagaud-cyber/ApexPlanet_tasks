# 🌐 ApexPlanet Web Development Internship (45 Days)

This repository contains all the projects completed during the **ApexPlanet Software Pvt. Ltd. Web Development Internship (45 Days)**.

The internship focuses on developing practical web development skills using **HTML5, CSS3, and JavaScript**, progressing from basic webpage creation to building complete responsive web applications.

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Visual Studio Code
- Git & GitHub

### ✅ Task 1 – Basics of Web Development
- Create a simple webpage using HTML
- Apply CSS styling
- Add JavaScript alert functionality

### ✅ Task 2 – Intermediate HTML, CSS & JavaScript
- Contact Form
- Form Validation
- Responsive Layout
- Dynamic To-Do List / Image Gallery

### ✅ Task 3 – Advanced Styling & JavaScript
- Responsive Design using Media Queries
- Interactive Quiz / Image Carousel

### ✅ Task 4 – Full Project Implementation
- Personal Portfolio Website
- To-Do App with Local Storage
- Product Listing Page with Filtering & Sorting

### ✅ Task 5 – Final Project & Optimization
- Complete Web Application
- Performance Optimization
- Cross-Browser Compatibility
- Mobile Responsive Design
